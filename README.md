### commit 컨벤션
<ul>
  <li>Feat – 새로운 기능 구현</li>
  <li>Fix – 버그 수정</li>
  <li>Mod – 코드 수정</li>
  <li>Ui - UI 수정</li>
  <li>Del – 필요 없는 파일 삭제</li>
  <li>Merge – 다른 브랜치와 merge</li>
  <li>Move – 파일이나 코드의 이동</li>
  <li>Rename – 이름 수정</li>
  <li>Docs – README와 같은 문서 수정</li>
</ul>
